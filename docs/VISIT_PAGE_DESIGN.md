## §0. 2단계 범위 확정

**2단계 = 방문 확정 + 시술 당일 평가(Phase 1) + 효과 평가(Phase 2) + 귀국 후 다운타임 관리**

평가(리뷰)는 2단계에 포함. 귀국 후 다운타임 알림도 2단계 안에서 처리.
**3단계는 별도 단계로 만들지 않는다.** 기능이 얇고 2단계 연장선이기 때문.

## §1. 방문 확정 화면

### 1-1. 상태 입력

```
┌──────────────────────────────────────────┐
│  방문 정보 입력                            │
│                                          │
│  병원명   [ 강남 OO피부과          ▼ ]   │
│  시술명   [ 울쎄라                 ▼ ]   │
│  방문일   [ 2026년 4월 21일        📅 ]  │
│                                          │
│  [ AI 조언 받기 ]   ← 위 항목 입력 시 활성화 │
└──────────────────────────────────────────┘
```

### 1-2. AI 조언 버튼 활성화 조건

같은 화면, 상태만 전환. 별도 페이지 아님.

| 상태 | 활성화 조건 | AI 조언 내용 |
|------|------------|-------------|
| **방문 전** | 병원명 + 방문일 입력 완료 | 시술 전 주의사항 (공복 여부, 자외선 차단제 금지, 복약 중단 여부 등) |
| **시술 후** | "시술 받았어요" 체크 또는 날짜 변경 입력 | 사후 주의사항 (다운타임, 귀국일까지 금지 행동, 관리법) |

## §2. Before 사진 촬영 — 리마인드 3회

유저가 before 사진을 찍어두지 않으면 나중에 before/after 비교가 불가능하다.
리마인드를 3번 한다. 찍으면 이후 알림 중단.

| 타이밍 | 방법 | 문구 |
|--------|------|------|
| 예약 확정 입력 직후 | 인앱 팝업 | "시술 전 사진을 지금 찍어두세요. 나중에 효과를 직접 비교할 수 있습니다." |
| 방문 전날 저녁 | 푸시 알림 | "내일 시술 전, 사진 꼭 찍으세요." |
| 당일 아침 | 푸시 알림 | "오늘 출발 전에 사진 한 장 찍어두세요." |

### Before 사진 팝업 화면

```
┌──────────────────────────────────────────┐
│  📸 시술 전 사진을 지금 찍어두세요          │
│                                          │
│  나중에 효과를 직접 비교할 수 있습니다.     │
│  저장은 내 기기에만. 공개 안 됩니다.        │
│                                          │
│  [ 지금 찍기 ]        [ 나중에 하기 ]     │
└──────────────────────────────────────────┘
```

"나중에 하기" 선택 시 → 다음 리마인드 타이밍에 재시도.

## §5. Before/After 사진 공개 구조

### 5-1. 공개 단계 정의

| 단계 | 범위 | 추가 보상 | 비고 |
|------|------|----------|------|
| 비공개 | 본인만 | 없음 | 기본값 |
| 부위 공개 | 시술 부위만 크롭, 앱 내 | +300pt | 외부 검색 차단 |
| 얼굴 공개 | 전체 얼굴, 앱 내 | +600pt | 닉네임만, 실명 없음, 외부 차단 |

**SNS 공유 보상 없음** — 공유 여부 확인 불가. 자발적 공유는 막지 않으나 보상 대상 아님.

### 5-2. "내 변화 기록" 슬라이더

- 앱 내 개인 화면에서 날짜별 사진 비교
- 슬라이더 직접 조작 후 공개 욕구 생길 때 바로 공개 버튼 노출
- 시술별로 묶어서 표시

## §8. 귀국 후 다운타임 관리

3단계 없음. 시술 완료 체크 시점부터 자동 시작. 별도 화면 아님 — 푸시 알림으로 처리.

### 8-1. 다운타임 알림 예시

```
D+0  (귀국 당일)
  "오늘은 격렬한 운동을 피해주세요."

D+1
  "시술 부위를 만지거나 문지르지 마세요."

D+3  (보톡스 기준)
  "보톡스 효과가 나타나기 시작할 시기입니다. 효과를 느끼고 계신가요?"

D+7
  "이제 가벼운 운동은 가능합니다. 사우나·찜질방은 아직 피하세요."

D+14 (보톡스 기준 Phase 2 타이밍)
  "효과 평가를 남겨주세요. +400포인트" → Phase 2 평가 링크
```

### 8-2. 시술별 다운타임 알림 스케줄

알림 내용·횟수는 `procedures.review_followup_days`와 별도로
`procedures.downtime_alert_schedule` (jsonb) 컬럼에 저장.

```json
{
  "D0": "격렬한 운동 금지. 시술 부위 손대지 마세요.",
  "D1": "세안 시 시술 부위 마찰 금지.",
  "D3": "효과가 나타나기 시작할 시기입니다.",
  "D7": "가벼운 운동 가능. 사우나는 아직 금지.",
  "D14": "Phase 2 평가 유도"
}
```

관리자 페이지에서 시술별 스케줄 편집 가능.

### 8-3. 다운타임 종료 알림

```
"울쎄라 다운타임이 끝났습니다. 🎉
이제 모든 활동이 가능합니다.
90일 효과 평가 알림은 별도로 드립니다."
```
