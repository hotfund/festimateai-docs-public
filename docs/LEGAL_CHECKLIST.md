## §1. 의료법 27조 — 의료알선 금지

### 1-1. 조항 요약

```
의료법 제27조 제3항:
누구든지 영리를 목적으로 환자를 의료기관이나
의료인에게 소개·알선·유인하는 행위를 하여서는 아니 된다.
위반 시: 3년 이하 징역 또는 3천만 원 이하 벌금
```

### 1-2. Phase 1 안전 전략

**FestiMate = 정보 제공 플랫폼. 알선 아님.**

```
가능한 것
  - 클리닉 정보 게시 (시술 종류, 가격, 위치)
  - 유저 리뷰 수집·게시
  - 시술별 다운타임·주의사항 정보 제공
  - AI 플래너 (일정 제안, 시술 후보 좁히기)
  - 클리닉 문의 버튼 (유저가 직접 클리닉에 연락)

불가한 것 (Phase 1)
  - 예약 대행
  - 수수료 수취
  - "이 클리닉에서 받으세요" 직접 권유
  - 단체 구매·공동구매 알선
```

### 1-3. 클리닉 자가등록 구조 (핵심 방어 논리)

FestiMate는 플랫폼. 클리닉이 직접 정보를 등록.
콘텐츠 생산자 = 클리닉. FestiMate = 게시 공간 제공.
네이버·유튜브가 광고주 대신 책임 안 지는 것과 동일 논리.

## §2. 외국인환자 유치업자 등록

### 2-1. 근거 법령

```
의료법 제27조의2:
외국인환자를 유치하려는 자는 보건복지부에 등록하여야 한다.
```

### 2-2. 등록 요건

```
- 자본금: 1억 원 이상
- 보험: 1억 원 이상 (의료사고 배상 책임보험)
- 국내 사무소 보유
- 보건복지부 신고
```

### 2-3. 등록 타이밍 — Phase 2 직전 (확정)

```
Phase 1: 미등록 (정보 제공만, 수수료 없음)

Phase 2 전환 직전:
  1. 투자 유치 또는 자체 자금으로 자본금 1억 확보
  2. 책임보험 1억 가입
  3. 보건복지부 유치업자 등록 신청
  4. 등록 완료 후 수수료 수취 시작
```

### 2-4. 수수료 상한 (등록 후 적용)

```
법적 상한:
  의원급:    30%
  병원급:    20%
  상급종합:  15%

FestiMate 적용 기준 (확정):
  의원급 클리닉 기준: 20% (법적 상한보다 낮게 설정)
```

Phase 2 이전에 클리닉에 수수료 약정 사전 안내 가능. 계약은 등록 후.

## §3. 의료광고 규제

### 3-1. 근거 법령

```
의료법 제56조·제57조:
의료광고는 사전 심의를 받아야 한다.
과장·허위·비교 광고 금지.
```

### 3-2. FestiMate 방어 구조

클리닉 자가등록 → FestiMate는 플랫폼 → 광고 주체는 클리닉.
FestiMate가 직접 작성하는 콘텐츠는 금지 표현 철저히 준수.

### 3-3. 절대 금지 표현 목록 (확정)

```
"안전합니다" / "부작용 없습니다"
"효과 보장" / "반드시 효과가 있습니다"
"최고의 의사" / "국내 최고"
"OO% 개선" / "완벽한 결과"
"일본 의사보다 실력이 뛰어나다"
"손기술이 좋다" (검증 불가)
Before/After 사진을 FestiMate 직접 광고에 사용
  (유저가 리뷰로 올리는 건 별개)
```

### 3-4. 사용 가능한 표현 (MARKETING_MESSAGING.md 참조)

```
"같은 시술을 10배 더 많이 경험한 의사"
"FestiMate는 6가지 기준을 모두 통과한 클리닉만 보여줍니다"
"외국인에게 다른 가격을 받는 클리닉은 즉시 제외됩니다"
"리뷰 4.8 이상을 유지하지 못하면 목록에서 탈락합니다"
```

## §4. 개인정보 보호

### 4-1. 적용 법령

```
한국: 개인정보보호법
일본: 個人情報保護法 (APPI) — 일본인 유저 대상이므로 적용
```

### 4-2. 수집 정보 분류

| 정보 | 민감도 | 보관 위치 |
|------|--------|----------|
| 이메일·닉네임 | 일반 | 서버 |
| 여행 일정·귀국일 | 일반 | 서버 |
| 시술 종류·날짜 | 민감 (의료 관련) | 서버 (암호화) |
| 클리닉 문의 내용 | 민감 | 서버 (암호화) |
| Before/After 사진 (비공개) | 민감 (생체정보) | 사용자 기기만 |
| Before/After 사진 (공개 선택 시) | 민감 | 서버 (암호화, 동의 후) |

### 4-3. Before/After 사진 원칙 (확정)

```
비공개 (기본):
  사용자 스마트폰 로컬 저장
  FestiMate 서버 전송 없음
  앱 삭제 시 함께 삭제

공개 선택 시 (부위/얼굴 공개):
  사용자 명시적 동의 후 서버 업로드
  암호화 저장
  닉네임만 표시, 실명 없음
  외부 검색 차단
```

### 4-4. APPI 준수 사항

개인정보처리방침에 아래 내용 명시:

```
- 수집 목적 명시
- 제3자 제공 금지 (클리닉 제외, 동의 시)
- 요청 시 삭제 의무
- 국외(한국) 이전 안전조치 명시
```

### 4-5. 개인정보처리방침 필수 항목

```
1. 수집하는 개인정보 항목
2. 수집 목적
3. 보유·이용 기간
4. 제3자 제공 여부
5. 위탁 처리 여부
6. 민감정보 처리 (의료 관련, 생체정보)
7. 정보 주체의 권리 (열람·수정·삭제)
8. APPI 준수 (일본인 유저 대상)
9. 담당자 연락처
```

## §5. 메이트 법적 지위

### 5-1. 관련 법령

```
관광진흥법 제38조:
관광통역안내사 자격증 없이 외국인에게
유료로 관광 안내 행위 금지.
```

### 5-2. Phase별 메이트 정책 (확정)

```
Phase 1: 무료 메이트만 허용
  - 메이트 활동 = 친구 만들기 (소셜), 가이드 아님
  - 수수료·참가비 없음
  - 관광통역안내사 자격증 불필요
  - 법적 리스크 없음

Phase 2: 유료 메이트 도입 (법적 검토 후)
  검토 사항:
  1. 관광통역안내사 자격증 보유 메이트만 유료 허용
  2. 또는 "참가비" 구조로 가이드 서비스와 구분
  3. 법무법인 자문 후 결정
```

### 5-3. Phase 1 메이트 활동 가이드라인

```
가능
  - 함께 밥 먹기, 카페, 쇼핑
  - 길 안내, 번역 도움
  - 무료 동행

불가 (Phase 1)
  - 유료 투어 진행
  - 참가비 수취
  - 공식 가이드 자격 없이 관광지 해설
```

## §8. 의료사고 대응 원칙

FestiMate는 의료행위를 하지 않으므로 의료사고 직접 책임 없음.
플랫폼으로서 신뢰도 관리 차원에서 대응 프로세스 필요.

```
Phase 1:
  - 클리닉 신고 기능 (clinic_reports 테이블)
  - 신고 접수 → 관리자 확인 → 필요 시 클리닉 목록 제외
  - FestiMate는 중개 책임 없음, 정보 제공자로서만 대응

Phase 2+:
  - B2B Claim Portal 운영
  - 책임보험으로 유치 관련 분쟁 대응
```
