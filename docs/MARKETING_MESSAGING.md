## 개요

일본인 관광객 대상 K-Beauty 서비스의 핵심 마케팅 메시지 정의.
"왜 한국에서 받아야 하나?" 질문에 대한 방어 가능한 표현 모음.

---

## 1. 핵심 가치 제안 (Core Value Proposition)

### 1-1. 한 줄 요약

```
"경험이 가장 많은 의사에게, 검증된 클리닉에서, 일본의 절반 이하 가격으로."
```

### 1-2. 세 가지 축

| 축 | 메시지 | 근거 |
|----|--------|------|
| **경험** | 강남 피부과 의사는 같은 시술을 일본 의사보다 10배 많이 경험합니다 | 한국 시술 건수 세계 1위, 강남 1개 클리닉 하루 20~30건 |
| **경쟁** | FestiMate에 등록된 클리닉은 가장 치열한 시장에서 살아남은 곳입니다 | 강남 피부과·성형외과 수백 개 경쟁, 6-gate 필터 통과 |
| **가격** | 항공+숙박 포함해도 일본에서 받는 것보다 저렴한 경우가 많습니다 | 시술비 1/3~1/10 수준 (시술별 상이) |

## 2. "왜 한국인가?" 메시지 상세

### 2-1. 경험·기술

**쓸 수 있는 표현:**
```
"같은 시술을 10배 더 많이 경험한 의사"
"반복이 만드는 기술 — 보톡스 1만 번과 1천 번은 다릅니다"
"최신 장비를 가장 먼저, 가장 많이 사용한 클리닉"
"세계에서 미용의료 경쟁이 가장 치열한 시장, 강남"
```

**쓰면 안 되는 표현:**
```
❌ "한국 의사가 더 잘한다"       → 주관적, 다른 나라 의사 비하
❌ "손기술이 좋다"               → 검증 불가, 오만하게 들림
❌ "일본 의사보다 실력이 뛰어나다" → 방어 불가
❌ "안전합니다"                  → 의료적 보증 불가
```

### 2-2. 검증·신뢰

**쓸 수 있는 표현:**
```
"FestiMate는 6가지 기준을 모두 통과한 클리닉만 보여줍니다"
"외국인에게 다른 가격을 받는 클리닉은 즉시 제외됩니다"
"리뷰 4.8 이상을 유지하지 못하면 FestiMate 목록에서 탈락합니다"
"일본어로 상담 가능한 클리닉만 등록됩니다"
```

### 2-3. 가격

**쓸 수 있는 표현:**
```
"보톡스 ¥8,000~ (일본인이라면 가격 차이를 아실 겁니다)"
"항공권+숙박+시술, 합산해도 이득인 경우가 많습니다"
```

**쓰면 안 되는 표현:**
```
❌ "일본보다 XX% 저렴"  → 기준 불명확, 클리닉마다 다름
❌ "최저가 보장"        → 허위 광고 가능성
```

### 2-4. 트렌드·최신성

**쓸 수 있는 표현:**
```
"울쎄라, 써마지, 리쥬란 — 일본보다 2~3년 먼저 대중화된 시술들"
"K-뷰티 트렌드의 출발점에서 직접 받으세요"
"인스타그램에서 본 그 시술, 한국이 원조입니다"
```

## 3. 타겟 페르소나별 메시지

### 3-1. 메인 타겟: 일본인 여성 20~40대

**페인 포인트:**
- 일본에서 받고 싶지만 너무 비쌈
- 어느 클리닉이 좋은지 모름
- 일본어로 상담이 안 됨
- 시술 후 귀국해도 되는지 불안

**FestiMate 메시지:**
```
"일본어로 상담 가능한 클리닉만 모았습니다"
"귀국일에 맞춰 받을 수 있는 시술을 AI가 골라줍니다"
"외국인이라고 바가지 없는 클리닉만 등록됩니다"
```

### 3-2. 서브 타겟: 한국 여행 + 뷰티 병행 계획자

**메시지:**
```
"여행 일정에 뷰티 스케줄을 더하세요"
"쇼핑·맛집·시술, 한 번의 한국 여행으로 전부"
"항공권 값은 이미 지불했습니다 — 시술 하나 더 추가해도 일본보다 저렴합니다"
```

## 4. 투자자용 메시지 (Investor Pitch)

### 4-1. 시장 규모 근거

```
일본 → 한국 의료관광 연 50만 명+
한국 미용의료 시장 5조 원 (2024)
일본인 1인당 평균 시술 지출 추정 ¥80,000~200,000
```

### 4-2. 경쟁 우위

```
강남언니: 한국인 대상, 일본어 없음
Creatrip: 여행 종합, 뷰티 깊이 부족
FestiMate: 일본인 특화 + 여행 일정 + AI 귀국일 역산 → 블루오션
```

### 4-3. 투자자 Q&A

**Q: 강남언니를 못 이기지 않나?**
```
강남언니는 한국인 앱입니다. 일본어 UI 없고, 귀국일 역산 없고,
여행 일정 통합 없습니다. 우리 유저는 다릅니다.
```

**Q: Creatrip이 이미 하고 있지 않나?**
```
Creatrip은 여행사입니다. 뷰티는 서브 카테고리입니다.
시술별 다운타임, 의사 정보, 금기 사항, 귀국일 역산 없습니다.
```

**Q: AI 기능은 따라 하면 되지 않나?**
```
AI보다 데이터가 진입장벽입니다. 클리닉 DB, 시술별 flight restriction,
OCR 이벤트 가격, 외국인 동일가 배지 — 이 데이터를 모으는 데 시간이 걸립니다.
```
