## 4. 핵심 차별화 요약 (투자자용 한 페이지)

### FestiMate만 가진 것 (3개의 Unique Feature)

```
1. 귀국일 역산 뷰티 스케줄
   "보톡스는 귀국 3일 전까지 가능합니다"
   → 일본인 여행객이 가장 두려워하는 "회복 중에 귀국" 리스크를 AI가 자동 제거

2. 여행 일정 + 뷰티 + 메이트 3결합
   → 강남언니는 뷰티 앱, Creatrip은 여행 앱. FestiMate는 두 가지를 + 한국인 메이트까지 연결
   → 경쟁사가 단일 도메인에 갇혀 있는 동안 FestiMate는 여행 전체를 커버

3. OCR 실시간 이벤트 가격
   → Instagram 배너 → GPT-4 Vision → 실시간 이벤트 가격 자동 수집
   → 경쟁사가 따라오기 가장 어려운 데이터 레이어
```

### 경쟁사 대비 포지셔닝 맵

```
            높음
            ↑ 여행 통합도
            |
FestiMate ──┼──────── (여행 HIGH + 뷰티 HIGH) [목표 포지션]
            |

            |
서울의료관광─┤ (여행 LOW + 뷰티 MEDIUM)
            |

            |
바비톡 ──────┤ (여행 NONE + 뷰티 MEDIUM)
            ↓
            낮음
←───────────────────────────→
낮음    뷰티 정보 깊이     높음
```

## 6. 투자자 Q&A 예상 답변

**Q. 강남언니가 1위인데 FestiMate가 이길 수 있나?**

강남언니는 "뷰티 정보 앱"이고, FestiMate는 "여행 전체 플랫폼"이다. 강남언니 유저는 시술 정보를 찾고 나서 여행 일정을 따로 짜야 한다. FestiMate는 그 두 가지를 하나로 합친다. 강남언니는 "어느 병원?"을 답하고, FestiMate는 "언제, 어떻게, 누구와 어떤 병원을?"을 통합 답한다.

**Q. Creatrip이 이미 여행 + 뷰티를 하고 있지 않나?**

Creatrip은 여행 예약과 뷰티 예약을 나란히 둔다. "통합"이 아니라 "병치"다. 귀국일 역산, AI 일정 생성, 한국인 메이트 연결은 없다. 뷰티 정보 깊이도 강남언니 대비 훨씬 얕다.

**Q. AI 기능을 경쟁사가 따라하면?**

OCR 이벤트 가격 파이프라인과 6-layer 키워드 온톨로지는 데이터 레이어가 핵심이다. 클리닉 데이터를 먼저 쌓은 쪽이 유리하다. FestiMate는 Phase 1에서 120~130개 고품질 클리닉을 수작업으로 큐레이션하고, OCR 파이프라인으로 경쟁자가 갖지 못한 실시간 이벤트 가격 데이터를 확보한다.
