# Mate 공급 확보 계획 (Korean Mate Recruitment Plan)

> **작성**: 2026-04-19
> **배경**: CEO Review에서 FestiMate 원안 유지 결정 후, 최대 실행 리스크로 식별된 "한국 Mate 공급" 문제의 실행 계획
> **단일 최대 리스크**: 일본 관광객이 와도 매칭할 한국 대학생 Mate 페어가 없으면 서비스 작동 불가

---

## 1. 문제 구조

### 왜 어려운가 — 5가지 구조적 원인

| # | 문제 | 왜 발생하나 |
|---|------|------------|
| 1 | 수익 없음 (Phase 1) | 플랫폼 수수료 0%, 메이트 보상은 "공짜 하루 + 친구 + SNS + 경력" — 비화폐 보상 |
| 2 | 페어 요구 (2인 1조) | 본인 + 친구 동시 설득 필요 → 한 명 빠지면 전체 무산 |
| 3 | 시간 투자 | 학적인증 + 오리엔테이션 + 6시간 세션 → 과외·알바 대비 매력도 낮음 |
| 4 | 일본어 요구 (권장) | 실질 접객 가능 수준 = 일본어과/관광학과 국한 |
| 5 | 수요 예측 불명확 | "매칭 요청 언제 올 지 모름" → 학생이 대기 비용 부담 |

### 공급 확보 6가지 접근 전략

- **[A] 학교 파트너십 (top-down)**: 학과장·지도교수 → 학점 연계. 지속 파이프, 느림.
- **[B] 학생 개별 모집 (bottom-up)**: 인스타·에브리타임. 빠름, 이탈률 높음.
- **[C] Founding Mate 20명**: 초기 공헌자 = "Founding" 타이틀 + 지분적 혜택. 강한 소속감.
- **[D] 추천제 (Mate→Mate)**: 추천인에게 포인트·티어. Cold start 후에 작동.
- **[E] 일본어과 직접 타겟팅**: 한국외대·부산외대·한양대 일어일문과.
- **[F] 일본문화 동아리 제휴**: JK 팬클럽·일본어 회화 동아리. 페어 조달 쉬움.

## 2. 할일 리스트 (우선순위 + 타임라인)

### Phase 0 (즉시, 이번 주~2주)

- [ ] **P0-1** 청주대 MOU 담당자 접촉 (이메일·전화 → 미팅 약속)
- [ ] **P0-2** 청주대 외 백업 학교 리스트업 5~7개교 (담당자 연락처 확보)
- [ ] **P0-3** Mate 모집 랜딩페이지 1차 시안 (`/mate-recruit` 라우트)
- [ ] **P0-4** Mate 자격·혜택 공식 1페이지 요약 (학생에게 보낼 자료)
- [ ] **P0-5** 법적 리스크 1차 체크: 학생 활동 = 근로인가? (변호사 30분 상담)

### Phase 1 (3~6주)

- [ ] **P1-6** 청주대 공식 제안서 작성 (PDF 5~8페이지)
- [ ] **P1-7** 일어일문과 + 관광학과 학과장 병행 접촉 (각 학교 3~5명)
- [ ] **P1-8** "Founding Mate 20명" 모집 캠페인 오픈 (인스타·에브리타임)
- [ ] **P1-9** 1차 면접 키트 (질문지·평가표·온보딩 자료)
- [ ] **P1-10** 오리엔테이션 커리큘럼 설계 (4시간 × 2회)
- [ ] **P1-11** Mate 공식 계약서 (NDA + 활동합의서, 변호사 감수)
- [ ] **P1-12** SNS 자산 키트 (인스타 템플릿·프로필 촬영 가이드)

### Phase 2 (6~12주)

- [ ] **P2-13** 1기 Mate 20명 온보딩 완료
- [ ] **P2-14** Mate 포털 `/mate-portal` 구현 (지원·페어·티어)
- [ ] **P2-15** 페어 파트너 초대 기능 (링크 기반)
- [ ] **P2-16** 추천제 (Mate→Mate 추천 포인트)
- [ ] **P2-17** Solo 대면 검증 프로세스 (1:1 인터뷰 SOP)
- [ ] **P2-18** 첫 3명 파일럿 고객 세션 진행 (일본 인플루언서)

### Phase 3 (3~6개월)

- [ ] **P3-19** 학교 수 5개 이상 확장 (활성 Mate 100명+)
- [ ] **P3-20** Verified 승급 시스템 자동화 (3회×평점 기반)
- [ ] **P3-21** Featured Mate 프로그램 (VIP 처우)
- [ ] **P3-22** Pro Guide 트랙 (유료 상품 권한 — Phase 2 연결)

## 3. 가장 중요한 3가지 (이것만 해도 80%)

1. **청주대 MOU 체결** — 안정적 공급 파이프 1개 확보
2. **Founding Mate 20명 캠페인** — 즉각 가동 가능한 시드 풀
3. **가치 제안 증명** — 첫 3명이 진심으로 즐거워하는 모습 콘텐츠화

## 4. 숨은 레버리지 채널 (무료~저비용)

| 채널 | 왜 효과적 | 비용 |
|------|----------|------|
| 한국외대·부산외대 일본학과 교수 소개 | 교수가 "외화벌이 체험학습" 프레임으로 권유 가능 | 식사비 |
| JLPT 학원 (시원스쿨·JLS 등) | 일본어 학습자 = 일본인 대화 기회 원함 | 제휴 |
| 주한 일본 유학생 교환 프로그램 | 반대편(한국→일본 유학) 네트워크 | 0 |
| 일본 문화원 한국 행사 자원봉사자 풀 | 이미 자발적 관심 입증됨 | 0 |
| 국제교류 동아리 (고려대 KILS, 연세대 YONSECO) | 페어 조달 쉬움 (동아리원끼리) | 0 |

## 5. 성공 지표

| KPI | 6개월 목표 | 12개월 목표 |
|-----|----------|-----------|
| 활성 Mate 수 | 40명 (2개 학교) | 150명 (5개 학교) |
| Verified 비율 | 10% | 30% |
| 평균 평점 | 4.5+ | 4.7+ |
| 페어 유지율 (3개월) | 70% | 85% |
| Mate → Mate 추천 비율 | 20% | 40% |

---

## 6. 본 계획과 사이트 완성의 관계

이 계획은 **사이트 완성 후** 본격 가동됩니다. 청주대 담당자를 만나려면:

- 동작하는 festimate.ai 도메인
- 한국어 홈 + Mate 소개 페이지
- 일본인 여행자 입장의 데모 경험

→ 사이트 완성이 선행 과제. 본 문서는 사이트 완성 후 1주 내 Phase 0 즉시 착수.
