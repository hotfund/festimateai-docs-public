# K-Beauty 경쟁사 분석 (일본 시장)

> **버전**: v0.1
> **작성**: 2026-04-20
> **목적**: 강남언니·바비톡 등 경쟁사 일본 서비스 현황 파악 → FestiMate K-Beauty 차별화 전략 수립
> **조사 방법**: 공개 자료, 뉴스, 일본어 리뷰, 공식 보도자료

---

## 1. 시장 규모 (배경)

| 지표 | 수치 | 출처 |
|------|------|------|
| 2024년 한국 방문 외국인 의료관광객 | **117만명** (역대 최대) | 보건복지부 |
| 그 중 미용 시술 비율 | **70%** (약 82만명) | 서울경제 |
| 그 중 일본인 비율 | **37만명** (전체의 32%) | 강남언니 공시 |
| 강남언니 통한 일본인 예약 | **16만건** (37만 중 43%) | 강남언니 |
| 일본인 의료관광 최대 강점 | 한국 가격 = 일본의 40~70% 수준 | 현지 리뷰 |

**결론**: 이미 거대 시장이다. 그리고 아직 아무도 "여행 일정 + 뷰티 통합"을 하지 않는다.

## 2. 강남언니 (UNNI) — 1위 압도적 선두

### 2-1. 일본 진출 타임라인

```
2019년 11월  일본 서비스 시작 (앱)
2020년 10월  일본어 앱 정식 출시
2022년       일본 매출 약 10억원
2023년       일본 매출 약 80억원 (8배 성장) → 전사 흑자 전환
2023년 12월  글로벌 'UNNI' 앱 출시 (영어 등 13개 언어)
2024년       전사 매출 530억원, 영업이익 128억원 (역대 최대)
2025년 2월   Series C 428억원 조달
2025년 9월   한일 누적 예약 130만건 돌파 (한국 100만 + 일본 30만)
2025년 10월  일본 제휴 병원 1,500개, 일본 유저 160만명
2025년 10월  중국·대만 UNNI 진출 → 6개 언어 서비스
```

### 2-2. 일본 서비스 기능 상세

| 기능 | 상세 |
|------|------|
| **언어** | 일본어 완전 지원 (앱·웹) |
| **리뷰** | 130만건 이상 (한일 합산), 사진 포함 |
| **채팅 상담** | 앱 내 클리닉과 직접 채팅 (자동번역 13개 언어) |
| **예약** | 앱 내 즉시예약 (일본 사용자 중 60%+ 사용) |
| **가격 정보** | 이벤트 가격 목록 표시 |
| **병원 탐색** | 시술명·지역·병원명 검색 |
| **UnniGuide** | 숙박·교통·통역·예약 가이드 서비스 (신기능) |
| **CCTV 필터** | 고스트 서저리 방지 (대리 수술 필터) |
| **오프라인** | 서울 강남 '언니가이드 센터' — 1:1 예약 가이드 현장 운영 |

### 2-3. 비즈니스 모델

```
수익원 1 (주): 광고비
  → 병원이 강남언니에 이벤트 페이지 개설
  → 사용자가 상담 신청 시 병원에 과금
  → 중소병원 월평균 100~200만원 광고비 지출

수익원 2: 즉시예약 수수료 (추정)
  → 예약 확정 시 거래액의 일정 % 수수료

수익원 3 (신규): UnniGuide 컨시어지
  → 유료 가이드 서비스
```

### 2-4. 강남언니의 약점 (= FestiMate 기회)

| 약점 | 구체적 문제 |
|------|-----------|
| **여행 일정 통합 없음** | 뷰티 예약만 됨. 숙박·관광·교통·축제와 연동 없음 |
| **다운타임 역산 없음** | "귀국 3일 전까지 보톡스" 같은 안전 일정 계산 불가 |
| **AI 플래닝 없음** | 검색+예약 기반. 예산·일정·피부 타입 기반 시술 조합 AI 추천 없음 |
| **여행 맥락 없음** | 한국 축제, K-pop, 로컬 경험과 뷰티를 묶는 서비스 없음 |
| **AI 번역 품질** | 자동번역 의존 → 의료 용어 뉘앙스 오류 가능 |
| **공장식 클리닉 과다** | 광고 기반이라 대형 프랜차이즈 위주 노출, 숨은 맛집 클리닉 발굴 약함 |
| **이벤트 가격 함정** | 광고 이벤트가 실제 상담 시 추가 비용으로 이어지는 경우 있음 |
| **메이트/가이드 없음** | 혼자 예약하고 혼자 방문. 현지 동행 서비스 없음 |

## 3. 바비톡 (Babitalk) — 후발, 2025년 8월 일본 진출

### 3-1. 일본 진출 현황

```
2025년 8월   일본 웹서비스 론칭 (첫 해외 진출)
             앱 없이 웹만 (web.babitalk.com/jp 형태 추정)
한국 유저:   900만명 (강남언니 대비 더 많은 가입자)
일본 진출:   강남언니보다 5년 늦음 — 완전한 후발주자
```

### 3-2. 일본 서비스 기능

| 기능 | 상세 |
|------|------|
| **언어** | 일본어 완전 지원 (웹) |
| **가격 비교** | 리프팅·히알루론산·보톡스·레이저·스킨부스터 등 이벤트 가격 목록 비교 |
| **병원 상담** | 병원 LINE 계정으로 상담 연결 (앱 내 채팅이 아닌 LINE 외부 연결) |
| **일본어 가능 병원 표시** | 아이콘으로 일본어 상담 가능 여부 표시 |
| **리뷰** | 일본어 리뷰 열람 (사진 포함) |
| **런칭 프로모션** | 상담 신청 시 올리브영 상품권 2,000엔, SNS 후기 시 추가 1,000엔 |

### 3-3. 바비톡의 약점

| 약점 | 상세 |
|------|------|
| **앱 없음** | 웹만 → 모바일 UX 열세 |
| **예약 시스템 없음** | 강남언니처럼 앱 내 예약 불가, LINE으로 외부 연결 |
| **리뷰 수 열세** | 강남언니 130만건 대비 일본 리뷰 데이터 절대적으로 적음 |
| **브랜드 인지도** | 일본에서 강남언니 이미 1위 고착화 |
| **강남언니와 동일 한계** | 여행 일정 통합, AI 플래닝, 다운타임 계산 없음 |

## 4. 기타 경쟁사

### 4-1. Seoul Guide Medical

- 한국 최대 정부 인증 의료관광 에이전시
- 성형·피부·치과·종합 병원 패키지
- 일본어 서비스 있음
- **약점**: 패키지 위주, 투명한 가격 없음, 소수 제휴 병원만

### 4-2. Jivaka Beauty (지바카)

- K-Beauty + K-Health 토탈 케어
- 의료기관·스파 협력 프로그램
- **약점**: 소규모, AI 없음, 플랫폼이 아닌 에이전시

### 4-3. HIMEDI

- 영어권 대상 의료 컨시어지
- "강남언니 vs UNNI" 비교 콘텐츠 제공
- **약점**: 일본어 미지원, 에이전시형

### 4-4. 언니가이드 센터 (강남언니 운영)

- 서울 강남구 논현동 오프라인 공간
- 외국인 전용 1:1 병원 예약 가이드
- **시사점**: 온라인만으론 부족 → 현장 서비스 수요 있음

## 5. 경쟁사 공통 한계 (FestiMate 차별화 포인트)

아무도 하지 않는 것:

```
1. 뷰티 + 여행 일정 통합
   강남언니: 예약 앱 (뷰티만)
   FestiMate: 뷰티 플랜 → [여행 일정에 추가] → 축제·관광·맛집과 하나의 일정표

2. 귀국일 역산 AI 스케줄
   강남언니: 유저가 직접 찾아봐야 함
   FestiMate: 귀국일 입력 → "보톡스는 내일까지만, 레이저는 3일 후부터 OOO 가능"

3. 예산·다운타임·피부타입 기반 AI 조합 추천
   강남언니: 시술 검색 → 유저가 선택
   FestiMate: AI가 "3일, ¥100k, 다운타임 1일 이내" → 최적 시술 3개 조합 추천

4. 현지 메이트와의 연결
   강남언니: 혼자 클리닉 방문
   FestiMate: 일본어 가능 메이트가 병원 동행 + 번역 + 이후 관광까지

5. 실시간 이벤트 가격 (OCR 파이프라인)
   강남언니: 광고 이벤트 페이지 (병원이 직접 올린 것)
   FestiMate: 인스타그램 이벤트 배너 자동 수집 → "오늘만 물광주사 40% 할인"

6. 다운타임 중 대안 활동 제안
   강남언니: 없음
   FestiMate: "레이저 후 2일 외출 자제 → 이 시간에 숙소 근처 카페 투어, 드라마 촬영지 실내 방문"
```

## 6. 일본인 유저 Pain Point (조사 기반)

현지 일본어 리뷰·조사에서 발견된 실제 불편함:

| Pain Point | 내용 | FestiMate 해결책 |
|-----------|------|----------------|
| **언어 장벽** | 상담 자동번역 오류, 의료 용어 뉘앙스 | 일본어 네이티브 큐레이션 + 메이트 동행 |
| **가격 불투명** | 이벤트가와 실제 상담가 차이 | 실제 이벤트 OCR 가격 + 범위 표시 |
| **다운타임 몰랐음** | 귀국 후 부기·멍 → "알았으면 다른 시술 선택" | 귀국일 기준 다운타임 역산 AI 스케줄 |
| **단독 방문 불안** | 의사 소통 + 혼자라 무서움 | 메이트 병원 동행 서비스 |
| **일정 분리 불편** | 뷰티 예약 따로, 관광 일정 따로 | 뷰티 플랜 + 여행 일정 통합 |
| **클리닉 선택 어려움** | 광고성 정보가 많아 진짜 괜찮은 곳 모름 | 큐레이션 + 일본어 가능 필터 + 리뷰 |
| **예약 후 불안** | 내가 예약한 의사가 실제로 시술하는지 | CCTV 필터 (강남언니 도입, 우리도 필요) |

```
강남언니: 클리닉 예약 플랫폼 (뷰티 특화)
바비톡:   가격 비교 + 후기 플랫폼 (뷰티 특화)

```

## 8. FestiMate K-Beauty 서비스 설계 수정 사항

경쟁사 분석 결과, BEAUTY_PAGE_DESIGN.md에 추가/수정 필요:

### 8-1. 즉시 추가해야 할 기능

| 기능 | 우선순위 | 이유 |
|------|---------|------|
| **클리닉별 LINE 연결** | 🔴 필수 | 바비톡도 사용, 일본인 LINE 사용률 85% |
| **일본어 가능 클리닉 배지** | 🔴 필수 | 강남언니도 핵심 필터 |
| **시술 후기 (일본어)** | 🔴 필수 | 강남언니 130만건 → 우리는 초기 큐레이션 선별 후기 |
| **이벤트 가격 표시** | 🟡 중요 | OCR 파이프라인이 경쟁 우위 |
| **귀국일 역산 스케줄** | 🟢 차별화 | 아무도 없음 |
| **여행 일정 통합** | 🟢 차별화 | 아무도 없음 |
| **AI 시술 조합 추천** | 🟢 차별화 | 아무도 없음 |

### 8-2. 강남언니 대비 초기 열세 인정 필요

- **리뷰 수**: 강남언니 130만 → 우리 0. Phase 1은 큐레이션 전략으로 보완 (적지만 믿을 수 있는 리뷰)
- **제휴 병원 수**: 강남언니 1,500개 → 우리 Phase 1 목표 120개. 포지셔닝: "일본어 대응 + AI 일정 통합 특화 엄선 클리닉"
- **브랜드 인지도**: 없음 → 초기는 니치 전략 (축제+뷰티 통합이라는 유일한 각도)

### 8-3. 강남언니가 못 하는 것으로 승부

강남언니는 의료 예약 플랫폼이기 때문에 구조적으로 하기 어려운 것들이 있다:
- 여행사적 요소 (축제·관광·숙박 연동)
- 현지 로컬 가이드 연결 (메이트)
- 비의료 뷰티 (퍼스널컬러, 헤어, 네일) 통합

이것이 FestiMate의 **법적으로도 안전하고 구조적으로도 유리한 포지션**이다.
