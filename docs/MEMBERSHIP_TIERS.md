# 회원제·등급 체계 (v1.0 초안)

> **작성**: 2026-04-14
> **상태**: 초안. 사용자 확정 대기
> **기반 원안**: 검색가능 → 반응가능 → 제안가능 (3단)

---

## 1. 철학

- **여행자 등급**과 **메이트 등급**을 분리 운영 (역할 다름)
- 3단 원안 유지 + **유료 구독/프로 가이드**는 부가 확장
- **Verified 이상만 매칭 요청 가능** → 안전·신뢰 핵심 라인
- **메이트 승급은 페어 단위** (두 명 모두 조건 충족해야 페어 승급)

## 2. 여행자 (Guest) 등급

| Lv | 등급 (가칭) | 가능한 활동 | 승급 조건 |
|---|---|---|---|
| **0** | **Visitor** | 🔍 검색·상품 열람 (무제한)<br>🤖 AI Guide **불가** | 비회원 |
| **1** | **Member** | ↑ + ❤️ 위시리스트<br>💬 호스트·메이트 문의<br>📝 리뷰 읽기 | 소셜 로그인 (Google / Kakao / LINE) |
| **2** | **Verified** ⭐ | ↑ + 🤖 **AI Guide 월 5회 무료** (초과 시 FP 소모)<br>🤝 메이트 매칭·모집 요청 (월 1회 무료)<br>📅 예약 진행<br>✍️ 리뷰 작성 | **여권 사본 제출** + 관리자 승인 |

> **Plus 구독 제거 (2026-04-16)** — 구독 모델 폐지. 모든 유료 사용은 **FestiPoint 크레딧**으로 통합. Verified 이상 사용자가 무료 한도 초과 시 FP 구매·소모 (ACCESS_CONTROL §4 참조).

### 핵심 룰
- **Verified 없이는 Mate 요청·예약·결제 불가**
- Plus는 선택. Phase 1에서 노출 여부 결정 필요.
- 한국인 게스트도 동일 체계 (외국인·한국인 무관)
- 19세 미만 가입 제한 (안전 정책)

## 3. 메이트 (Mate) 등급

| Lv | 등급 (가칭) | 가능한 활동 | 승급 조건 |
|---|---|---|---|
| **0** | **Applicant** | 대기 중, 활동 불가 | 학적 증명 제출 완료 |
| **1** | **Rookie** | 🟢 무료 세션 호스팅<br>📝 From Mate 모집글 작성 | 페어 승인 + 오리엔테이션 수료 |
| **2** | **Verified** ⭐ | ↑ + 💰 **유료 상품 제안 가능**<br>✉️ From Tourists 요청에 응답 가능 | **5회** 세션 완료 × 평점 4.5+ (자동 승급) |
| **3** | **Featured** | ↑ + 🌟 플랫폼 추천 노출<br>📸 대표 메이트 페이지 등재<br>💸 수수료율 협상권 | 20회 + 평점 4.8+ + 다국어 인증 |
| **4** | **Pro Guide** | 단독 활동<br>B2B 송출 (오프라인 한일 송출 연계)<br>전문 가이드 자격증 | **30회 + 평점 투표 20개 + 관리자 최종 심사** (자동 불가) |

> **v1 UI 단순화 (2026-04-20)**: Matching 페이지 카드 배지는 RKY/VER/PRO 3단으로 표시 (Featured는 VER로 병합 표시). 내부 등급은 4단 유지, 표시만 단순화.

### 핵심 룰
- **페어 단위 승급** — 페어 2명 모두 조건 충족 시만 승급
- **페어 타입 (2026-04-15 재설계)**: Phase 1 FF/MF → Phase 2 + GG/BB (Rainbow Circle) → Phase 3 1:1 심사제. ~~MM 페어 금지~~ 번복됨 (상세: ACCESS_CONTROL.md §5)
- From Tourists 요청 응답은 **Verified 이상만** (게스트 신뢰 보호)
- 평점 4.0 미만 연속 2회 → 일시 정지
- 노쇼 3회 → 강등 / 활동 정지
- 오리엔테이션 미수료 → 활동 불가
- 학적 만료 (졸업·휴학) → 상태 업데이트 필요
- **야간·음주·숙박 제한 없음** (2026-04-15 결정: 성인 자율, 권고 사항으로만 표시)

## 4. 공급자 (Supplier) 등급 — KBeauty · KMaster · Templestay

| Lv | 등급 | 가능한 활동 | 승급 조건 |
|---|---|---|---|
| **0** | **Pending** | 검수 대기 | 가입·사업자인증 완료 |
| **1** | **Verified** | 상품 업로드·판매 | 관리자 승인 |
| **2** | **Preferred** | ↑ + 검색 상단 노출<br>수수료율 협상 가능 | 평점 4.8+ × 20건 완료 |
| **-1** | **Restricted** | 일시 정지·숨김 | 평점 하락 or 분쟁 반복 |

### 핵심 룰
- Phase 1 수수료율 일괄 20%
- Preferred 티어 진입 시 19%로 인하 검토 가능
- 분쟁 발생 건은 관리자 검토 후 Restricted 결정

## 5. 원안(3단) → 현재안 매핑

```
원안:       검색가능   →   반응가능   →   제안가능
               │              │              │
게스트:     Visitor(0)   →   Member(1)   →   Verified(2)
                                          + Plus(3, 유료옵션)
메이트:     Applicant(0) →   Rookie(1)   →   Verified(2)
                                          + Featured(3)
                                          + Pro Guide(4, 미래)
공급자:     Pending(0)   →   Verified(1) →   Preferred(2)
```

→ 3단 원안의 **철학 유지**, 세부 권한만 역할별로 다듬음.

### Verified 계층 (본인 인증) — 여권 전용

- **유일한 수단: 여권 사본 업로드 + 관리자 수동 승인**

- 사유: 매칭(대면 만남) 안전 보장에 정부 발급 신분증 수준 필요
- 추후 자동 검증(OCR + 안면 대조) 도입 시 관리자 수동 승인 해제 검토

**개인정보 저장**: 인증 확인 후 원본 문서는 **암호화 저장**, 필요 최소 정보만 메타 보관.
