## 2. 이 세션에서 확정·수정된 결정

### 2-2. AI 모델 기본값 하향 조정 (이전 CLI 제안 폐기)

**근거**: FestiMate 유스케이스 재분석 결과, 고급 추론이 필요한 케이스가 거의 없음. 대부분 RAG·검색·템플릿 채우기.

| 유스케이스 | 실제 작동 원리 | 필요 모델 |
|---|---|---|
| 여행 플랜 생성 | DB retrieval + 시간·이동 제약 + 일정 템플릿 | mini + 알고리즘 |
| 채팅 Q&A | 검색 + 자연어 포맷팅 | mini |
| 번역 JP↔KR↔EN | 번역 | mini / Gemini Flash / 전용 번역 API |
| 이미지 번역 (OCR) | OCR + 번역 | mini vision |
| 메이트 매칭 | 벡터 유사도 | **LLM 불필요** (pgvector) |
| 첫 메시지 초안 | 템플릿 + 변수 치환 | 템플릿 + 번역 |
| 여행 중 리플랜 | 제약 재충족 | mini (고급 추론 필요한 유일한 케이스) |

**수정된 기본값**:
```
ai.model.plan      = "gpt-4o-mini"
ai.model.chat      = "gpt-4o-mini"
ai.model.translate = "gpt-4o-mini"  # 출시 후 DeepL/Gemini Flash 비교
ai.model.vision    = "gpt-4o-mini"
```

**비용 재추산** (이전 대비 1/6):
- 1K MAU: ~$6/월
- 10K MAU: ~$60/월

**정책**: 실제 품질 로그로 판단, 필요시 config에서 모델 상향 (코드 변경 없음).
